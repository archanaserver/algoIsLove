* Firstly break down the problem and note down the input, output, constraints & all edge cases.

* Analyze the solution and find it's memoizing i.e. overlapping part.
* Break down your problem with the Top-Down Dynamic Approach(recursion + memoization) because it optimizes space and time complexity.
* Now, turn your solution with the Bottom-Up Dynamic Approach(using iteration) because that saves memory cost.
* That's it! You'll get the most optimal solution.
